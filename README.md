# Tibbiy Malaka - Medical Training Center Website

Professional website for "Xodimlar Malakasini Oshirish va Ularni Qayta Tayyorlash Ilm Dargohi" - Medical Training Center in Uzbekistan.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## Getting Started
```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
